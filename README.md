# SmallBasket

An E-Commerce platform with Review System Implementation

## Getting Started

### Tech Stack

* ReactJS
* NodeJS
* MongoDB
* AWS

### Instructions to Execute

#### Server

```
$ git clone https://github.com/vinayakguptaa/SmallBasket
$ cd SmallBasket/Server
$ touch .env 
>>> Refer .env.example for environment variables
$ npm install
$ npm start
```
